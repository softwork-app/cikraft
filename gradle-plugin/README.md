# Module gradle-plugin

The Gradle plugin creates an extension to define your IFlows, Proxies and stages in your Gradle script to automate your deployment.

> [!IMPORTANT]
> The Gradle plugin uses the experimental [Declarative DSL](https://declarative.gradle.org/) and the `jvmApplication` from https://github.com/Kotlin/declarative-gradle-jetbrains-ecosystem-plugin
